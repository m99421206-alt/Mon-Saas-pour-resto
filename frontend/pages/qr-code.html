Votre QR Code

Imprimez ce QR code et placez-le sur vos tables. Vos clients n'auront qu'à le scanner pour voir votre menu.

Pour tester avec un téléphone : l’URL 127.0.0.1 ou localhost ne fonctionne pas sur le mobile — c’est son propre appareil, pas votre PC. Dans frontend/js/config.js, renseignez PUBLIC_SITE_ORIGIN avec l’adresse réseau de votre ordinateur (ex. http://192.168.1.10:5500), même Wi‑Fi, puis régénérez le QR code. Lancez aussi l’API sur ce réseau (port 4000 écoutant sur votre machine).
